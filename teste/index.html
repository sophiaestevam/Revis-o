Revisão de repetição

  • O que e uma estrutura de repetição?
  • Uma estrutura de repetição é um bloco de código que se repete enquanto uma condição é verdadeira.
  • Para que serve ?
  • Serve para automatizar tarefas que precisam ser repetidas várias vezes, economizando tempo e esforço.
  • while - inicio (antes de entrar no bloco de codigo) - pre - 0 + vezes
  • do-while - final (apos o bloco de codigo) - pos - 1 + vezes
  • for - ideal para cenarios onde: a) conheço o numero de repetição: b) possuo variaveis de controle/contadores
